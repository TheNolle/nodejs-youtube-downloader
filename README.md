## Installation ğŸ”§
1. Clone the repository:
```bash
git clone https://github.com/thenolle/nodejs-youtube-downloader.git
cd nodejs-youtube-downloader
```
2. Install dependencies:
```bash
# if using npm
npm install
# if using pnpm (recommended)
pnpm install
```
3. Build and run the application:
```bash
# if using npm
npm run build
npm start
npm run server
# if using pnpm
pnpm build
pnpm start
pnpm run server
```
